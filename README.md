Программа обрабатывает результаты нейросети из выбранной папки, классифицирует линии как `STRAIGHT` или `CURVE`, отслеживает их между кадрами и выделяет осевую линию `AXIS`.

## Что делает программа

- читает координаты линий из txt-файлов;
- строит прямую методом наименьших квадратов;
- считает RMSE отклонения точек от прямой;
- отбрасывает выбросные точки нейросети;
- определяет тип линии: прямая или кривая;
- отслеживает линии между кадрами;
- сохраняет размеченные изображения в новоую папку.
